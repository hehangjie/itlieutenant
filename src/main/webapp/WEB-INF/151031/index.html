初识篇《My Initial J》

1/3


在车站的时候，还来不及再拉一次J的手，公车就匆匆来到。街景开始倒退的时候，我连打开车窗的时间都不够。只能把五官贴在车窗玻璃看她挥手的身影渐远。
2/3

        然后是空荡荡的乡村公路，跑着一辆只有一位乘客的空车。心里一下落空了，每一次感受空气的清新，就会想起江镜的红土地。每一次对红土地的依恋都会在夕阳到来之前失落。就像见不了阳光的鬼魂，每次醒来总是身在钢筋混凝土的城市中。J 说不喜欢高楼，不喜欢拥挤的城市。我才发现，我并不是不讨厌拥挤，而是早就从厌烦被熬到了麻木。所以我觉得我是个蠢人，这么大了竟然不懂得生活。
3/3

        也感谢上帝，让我遇到了J。上帝说，这小伙子不错，试着把交手给他。在那家超市门口，准确的说那是一家豪华版的小卖部，我转过身的一刹那。上帝说，要有光。于是我的世界就有了光。长期在黑暗里痛苦抽筋成长的我，猛然间就给闪到了。闪得夜如白昼，接着是长达数日的失眠。那是我失眠最长的一段日子，我开始怀疑是不是以后都不用睡觉了。我知道，接下来我会生病。于是就生病了，这才开始正常睡觉。我还知道，这病不用看也会好，于是睡够了病就好。

        我知道，还有一小段路要走，我还会再生病。

        然，现在只想要再多见一次，再多牵一次手。我知道，暂时的离别，是为了最终的天长地久。

        我知道，她最终留下的地方，就是那块可以让我尘埃落定的土地。

--写于2011年《My Inital J》
